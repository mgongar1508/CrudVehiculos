# Crud 2, dos tablas con Login

Esta es una aplicación hecha en php con las librerias Dotenv y faker para la gestión de vehiculos.

## Funcionamiento

La aplicación cconsiste en una página de login que autentica usuarios y después de logearse,
le muestra la lista de todos los vehiculos. El usuario podrá crear vehiculos y editar/borrar los
vehiculos que le pertenecen.
